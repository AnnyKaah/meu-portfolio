Bem-vindo ao repositório do meu portfólio pessoal! Este projeto não é apenas uma vitrine, mas uma experiência imersiva desenvolvida com **React, Framer Motion e WebGL**. Aqui, apresento minha transição de Engenheira de Materiais para Desenvolvedora Full Stack, combinando pensamento sistêmico com código de alta performance.

---

## 🌌 Sobre o Projeto

Este portfólio foi construído para ser **rápido, responsivo e visualmente impactante**. Ele utiliza um tema espacial "Cyberpunk/Neon" com partículas interativas e animações fluidas.

### ✨ Destaques Técnicos
- **🎨 UI/UX Imersiva:** Design System próprio com *Glassmorphism*, *Neon Glow* e tipografia moderna.
- **⚡ Performance:** Construído com Vite para carregamento instantâneo.
- **🎭 Animações:** Uso avançado de `Framer Motion` para transições de página, *scroll reveal* e micro-interações.
- **🎮 Gamificação:** Seção dedicada a jogos desenvolvidos com Canvas API e Web Audio.
- **📱 PWA Ready:** Responsividade total para Mobile, Tablet e Desktop.

## 🚀 Como Rodar Localmente

Quer ver a mágica acontecer na sua máquina? Siga os passos:

```bash
# 1. Clone o repositório
git clone https://github.com/AnnyKaah/meu-portfolio.git

# 2. Entre na pasta
cd meu-portfolio

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` e aproveite a viagem! 🚀
